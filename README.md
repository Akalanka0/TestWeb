In the project folder, run these commands:

npm start
Starts the app in development mode at http://localhost:3000. The page reloads on code changes.

npm test
Runs tests in watch mode.

npm run build
Builds the app for production in the build folder with optimized performance.

npm run eject
Removes the Create React App build dependency to customize configuration files.
Note: This is irreversible.
